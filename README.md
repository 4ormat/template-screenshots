# template-screenshots

Utility for taking website screenshots at different viewports.


## Setup

### Install Dependencies
```bash
nvm use
npm install
```

### Add API Keys

```
touch .env
```

.env file should look like
```
CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

## Usage

```bash
./bin/index.js
```
